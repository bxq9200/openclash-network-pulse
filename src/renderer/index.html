如何找到控制面板地址和密钥

进入 OpenClash 的运行状态页，右侧“控制面板”卡片里可以复制地址和密钥。

  1. 打开 OpenWrt / 路由器后台，进入 Services → OpenClash。
  2. 切到“运行状态”，找到右侧的“控制面板”。
  3. 复制绿色地址，例如 192.168.9.1:9090，填入路由 IP 和 API 端口。
  4. 点锁形按钮复制密钥，粘贴到 Secret。也可以直接粘贴 Bearer 文本，应用会自动识别。
